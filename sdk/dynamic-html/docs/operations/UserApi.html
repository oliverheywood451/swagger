list

URL

https://api.ordercloud.io/v1/buyers/{buyerID}/users

HTTP Method

GET

Response Type

Object

Parameters

  • Path: buyerID String

    ID of the buyer.

  • Query: userGroupID String

    ID of the user group.

  • Query: search String

    Word or phrase to search for.

  • Query: searchOn String

    Comma-delimited list of fields to search on.

  • Query: sortBy String

    Comma-delimited list of fields to sort by.

  • Query: page Integer

    Page of results to return. Default: 1

  • Query: pageSize Integer

    Number of results to return per page. Default: 20, max: 100.
